## 2. Phase 0.1 — 測定点変更とファイル名統一

### 2-1. 測定点の変更内容

| 変更前 (v1) | 変更後 (v2) |
|------------|------------|
| 中心        | 室戸沖      |
| 北          | 北西        |
| 南          | （廃止）    |
| 東          | 東（維持）  |
| 西          | 西（維持）  |
| （新規）    | 北東        |

**変更理由：** 室戸岬は南方向に突き出た尖った地形。「南」地点は岬の陸地に近く、海流データとして意味が薄い。代わりに北西・北東を追加し、室戸岬沖の広い範囲をカバーする配置に変更。

### 2-5. 発生した問題と解決

**問題①：バッチファイルの文字化け**  
- 原因：Pythonで`open(..., 'w', encoding='utf-8')`でバッチファイルを書いたが、Windows CMDはcp932（Shift-JIS）で解釈するため文字化け
- 解決：Pythonで`encoding='cp932'`、改行`\r\n`（CRLF）で書き直し。さらにバッチファイル内の日本語を完全除去してASCII文字のみに統一

**問題②：フォルダ名変更（mv）の失敗**  
- 原因：OneDriveマウント上では`mv jcope_muroto muroto_offshore_current`が「Permission denied」
- 解決：`cp -r jcope_muroto muroto_offshore_current`でコピー後、元フォルダはWindowsエクスプローラから手動削除を案内
- 現状：`jcope_muroto`フォルダはまだ残存（Windowsエクスプローラから削除可能）

### 3-3. カスタム期間ピッカー（JS核心部分）

```javascript
let customStartDate = null;
let customEndDate   = null;

function applyCustomRange() {
    const s = document.getElementById('customStart').value;
    const e = document.getElementById('customEnd').value;
    if (!s || !e || s > e) { alert('正しい期間を入力してください'); return; }
    customStartDate = s;
    customEndDate   = e;
    currentDays     = 0;
    document.querySelectorAll('.btn-days').forEach(b => b.classList.remove('active'));
    renderCharts();
}

function getFiltered() {
    if (customStartDate && customEndDate) {
        return allData.filter(r =>
            r.point === currentPoint &&
            r.date >= customStartDate &&
            r.date <= customEndDate
        ).sort((a,b) => a.date.localeCompare(b.date));
    }
    const allDates = uniqDates();
    const show = new Set(allDates.slice(-currentDays));
    return allData.filter(r => r.point === currentPoint && show.has(r.date))
                  .sort((a,b) => a.date.localeCompare(b.date));
}
```

### 3-5. 発生した問題と解決

**問題：ダッシュボードが真っ白（blank page）**  
- 原因：ワークスペースのHTMLファイルが行799で切り捨てられており、`window.addEventListener('load', autoLoad)` とクロージングタグが欠落していた
- 発見方法：`grep -c '</script>'` でタグ数を確認（2個 vs 本来3個）、行数確認（817行 vs 正常864行）
- 解決：欠落していた約50行をファイル末尾に追記
