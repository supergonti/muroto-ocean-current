## 1. プロジェクト概要

室戸岬沖（高知県）の海流・水温・塩分データを毎日自動収集し、ブラウザで閲覧できるダッシュボードとして公開するシステムの開発プロジェクト。

- データソース：CMEMS（Copernicus Marine Service）
- ライブラリ：`copernicusmarine`（Python）
- ダッシュボード：Chart.js v4.4.0（静的HTML+JS）
- 自動収集：Coworkスケジューラー（毎朝6時）

## 4. 次にやること：Phase 1 — GitHub移行 + Web公開

### 目標
ダッシュボードをGitHub Pages経由でインターネット公開し、URLでアクセスできるようにする。

### GitHubアカウント状況
- アカウント：あり
- git コマンド：なし（インストール不要）
- 方針：**GitHubウェブブラウザからアップロード**

### アップロードするファイル（2ファイルのみ）

| ファイル | 説明 |
|---------|------|
| `index.html` | ダッシュボード（GitHub Pages用）|
| `muroto_offshore_current_dashboard_data.js` | データJS（630KB）|

Pythonスクリプト・バッチファイル・CSVはローカルPC専用のため不要。

### 手順（次チャットで実行）

**Step 1：GitHubリポジトリ作成**
1. github.com にログイン
2. 「+」→「New repository」
3. 名前：`muroto-ocean-current`、Public、READMEなし（空）で作成

**Step 2：ファイルをアップロード**
1. リポジトリトップ → 「uploading an existing file」をクリック
2. 以下2ファイルをドラッグ＆ドロップ：
   - `index.html`
   - `muroto_offshore_current_dashboard_data.js`
3. 「Commit changes」ボタンをクリック

**Step 3：GitHub Pagesを有効化**
1. リポジトリの「Settings」タブ
2. 左メニュー「Pages」
3. Source：「Deploy from a branch」
4. Branch：`main` / `/(root)` → 「Save」
5. 数分後に `https://（ユーザー名）.github.io/muroto-ocean-current/` が発行される

**Step 4：動作確認**
- 発行されたURLをブラウザで開いてダッシュボードが表示されることを確認

### 注意事項
- `index.html` 内の `<script src="muroto_offshore_current_dashboard_data.js">` のパスは変更不要（同じルートに両ファイルを置くため）
- GitHub Pages が有効になるまで最大5〜10分かかる場合がある

## 7. 技術メモ（次チャットのClaudeへ）

- HTMLファイルは `file://` プロトコルで動作確認済み（CORS問題なし）
- ダッシュボードのデータ読み込みは `window.MUROTO_CSV_TEXT` グローバル変数でJS内に埋め込む方式
- Windowsのバッチファイルは **cp932エンコーディング + CRLF改行** で作成すること（UTF-8だとCMDで文字化け）
- OneDriveマウント上のフォルダ削除は権限エラーになることがある → `cp -r`でコピー後、Windowsから手動削除
